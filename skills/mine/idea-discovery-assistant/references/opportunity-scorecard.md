# Opportunity Scorecard

Score each candidate from 1 to 5:

- Problem clarity: can the problem be stated without vague buzzwords?
- User clarity: is there a specific user or actor?
- MVP feasibility: can a useful prototype be built in days/weeks?
- AI leverage: does IA improve capability, workflow, or quality?
- Data availability: can inputs be obtained safely?
- Evaluation: can outputs be tested or scored?
- Learning value: does it exercise applied AI engineering skills?
- Risk: privacy, cost, ambiguity, external dependencies.

Prefer ideas with high clarity, high evaluation potential, and manageable risk.
